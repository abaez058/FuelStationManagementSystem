# TurboFill Fuel Station Management System  
*A C-based Terminal Application for Daily Fuel Sales & Administrative Operations*

## Overview

TurboFill is a full-featured **Fuel Station Management System** written entirely in **C**, designed to simulate real-world fuel station logic including:

- Customer fueling operations  
- Fuel pricing + tax calculation  
- Loyalty and bulk discount programs  
- Inventory deduction and live stock warnings  
- Daily summary analytics  
- Administrative station monitoring  
- Receipt generation for each transaction  

This program runs in the terminal and uses nested menus to simulate real station workflows.

---

### **Daily Operations**
- Serve multiple customers  
- Select fuel type (Regular, Premium, Diesel)  
- Enforce max fueling limits  
- Vehicle type validation (motorcycle/car/truck)  
- Compatibility rules (e.g., trucks must use diesel)  
- Automatic tax calculation per vehicle type  
- Bulk discounts (30+ gal = 5%, 50+ gal = 10%)  
- Loyalty discount (additional 2%)  
- Generates a professional receipt  
- Updates real-time fuel stock
